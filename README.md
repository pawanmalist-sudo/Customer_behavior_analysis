# Customer Behavior Analysis 

## Overview

This project focuses on analyzing customer behavior to uncover meaningful insights that can help businesses improve decision-making. It demonstrates an end-to-end data analytics workflow, including data loading, cleaning, exploratory data analysis (EDA), SQL-based analysis, and data visualization. The final insights are presented through a Power BI dashboard, a detailed report, and a presentation created using Gamma.

---

## Dataset

The dataset contains customer-related information such as:

* Customer demographics
* Purchase history
* Product categories
* Review ratings
* Transaction details

This data is used to analyze patterns in customer behavior, preferences, and spending trends.

---

## Tools & Technologies

* **Python** (Pandas, NumPy, Matplotlib, Seaborn) – Data processing and EDA
* **PostgreSQL** – SQL queries and data analysis
* **Power BI** – Dashboard and visualization
* **Gamma** – Presentation (PPT) creation
* **Jupyter Notebook** – Development environment

---

## Steps Performed

### 1. Data Loading

* Imported dataset using Pandas
* Explored structure, columns, and data types

### 2. Data Cleaning

* Handled missing values using `isna()`, `fillna()`, and `dropna()`
* Removed duplicates and inconsistencies
* Converted data types for accurate analysis

### 3. Exploratory Data Analysis (EDA)

* Analyzed customer purchase patterns
* Visualized trends and distributions
* Identified correlations between variables

### 4. SQL Analysis (PostgreSQL)

* Loaded cleaned data into PostgreSQL
* Performed queries using:

  * Aggregations (`AVG`, `SUM`, `COUNT`)
  * Window functions (`ROW_NUMBER`, `RANK`)
  * Grouping and filtering
* Extracted insights like top customers, popular products, and spending trends

### 5. Dashboard Development (Power BI)

* Built interactive dashboards with key KPIs
* Included filters and slicers for dynamic analysis
* Visualized customer trends and product performance

### 6. Reporting & Presentation

* Created a structured report summarizing findings
* Developed a presentation using Gamma
* Highlighted insights and business recommendations

## How to Run

1. Clone the repository
2. Install required libraries:

   ```bash
   pip install pandas numpy matplotlib seaborn
   ```
3. Open the Jupyter Notebook and run all cells
4. Import the dataset into PostgreSQL and execute SQL queries
5. Open the Power BI file to explore the dashboard
6. Review the report and presentation for insights
